# Database System of College Students Restaurant based on Python GUI Library Tkinter
---

## Introduction

* This is a junior-year computer science student's experimental course project, which involves developing a database system with visualization requirements.
* Built with Python + MySQL.
* The database was initially connected to Huawei Cloud GaussDB, which has now expired. You can modify it to connect to your own local database.

---

## Software Architecture

* The system features one main page with multiple sub-pages.
* Navigation is done through clickable icons.

---

## Usage Instructions

* The code is located in the `src` folder.
* `main.py` is the main page.
* You need to modify the database connection settings in the code before generating the `.exe` file.
* The images used in the code have not been uploaded.
* A video demonstration of the working `.exe` file is available on Bilibili at:
    * Link: https://www.bilibili.com/video/BV1hK4y1S7Rv/
